ICÔNES PWA — ScrimSync
======================

Placez vos icônes PNG dans ce dossier :

  icon-192.png       — 192x192 px  (icône principale Android)
  icon-512.png       — 512x512 px  (icône haute résolution)
  apple-touch-icon.png — 180x180 px  (icône iPhone/iPad)
  maskable-icon.png  — 512x512 px  (icône avec safe zone pour Android)

Pour la maskable icon :
  Le logo doit tenir dans un cercle centré représentant 80% de l'image.
  Les 10% de chaque côté sont le "padding safe zone".
  Fond uni recommandé : #09090b (même noir que l'app).

Source de vos icônes :
  L'icône ScrimSync (S + grille calendrier) vous a été fournie en PNG.
  - Sauvegardez l'icône carrée comme icon-512.png
  - Redimensionnez-la en 192x192 pour icon-192.png
  - Redimensionnez-la en 180x180 pour apple-touch-icon.png
  - Pour maskable-icon.png, ajoutez 10% de padding sur fond #09090b

Outil recommandé pour redimensionner : https://squoosh.app
Outil pour générer maskable : https://maskable.app
